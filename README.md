# Dashboard-ARM
A dashboard for tracking CS papers and offer relevant analytics about them or their subject


# Minimum requirements for running the project on a non-ARM device, which has to simulate the architecture:
RAM: 10GB with running 8 async scrapers -> 5GB browsing the project
Storage: ~20GB for storing :
~10k papers (abstracts, not full papers) 
KeyBERT model
DistilBART model

# Start
Run the ```docker-compose up --build``` command to build the image
